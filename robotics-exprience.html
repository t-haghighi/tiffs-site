  • Intro to enginerring principals

    • The basic engineering cycle (I find it more of a thought process) every person going into any kind of engineering will become accustomed to and live by as they spend more time
    building things. I had zero expierence with any kind of building before and learning the engineering cycle and how to process thoughts differently was a challenge.
    Of course the only way to get around this challenge is to follow the engineering cycle (aka learning Mechanical Principals, forces, how to use gears, gear trains, etc.),
    ask for help, take notes (writing what happend in a language that makes sense to you, drawing pictures, color coating, etc.)and more interaction with your project that your working on.

  • Gear trains

    • Gears can be used to make the fast object like a drag racer or something with more pulling power such as a tractor.
    Personally I haven't mastered the complete ideology of using complex gear trains but understand the basic concept.
    Faster movement you would want to have the driver gear to be larger and the follower gear to be smaller creating a 2:1 ratio or larger (the ratio can be found by counting the number of teeth and inputting it in thr formula: n2/n1, n being the number of teeth).

  • Object manipulations

  • lessons learned

    • Three core lessons I learned from this program were:

    1. How to use the engineering cycle in my thought process to work smarter not harder.
    2. The importance of writing everything down. Each student was required to wrtite down their steps, successes and failures in a notebook and what I found is that writing drawing or describing more was better, we were told to write out steps and drwaings so that someone else could build the same exact robot.
    3. How to visualize realistically. Many ideas seemed to be perfect when I imagined them but in the process of building there was some kind of un forseen issue. After these expierences I would write down any possible errors and give myself extra time in case any problems arose.
